# KeyJinX Password Manager

Browser-based password vault with localStorage persistence.

## Features
- Add/view/delete passwords
- Copy to clipboard (website/username/password)
- Password masking for security
- Responsive neon-themed UI

## Demo
Open `index.html` in any browser.
